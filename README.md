# Runner's Hi Android Repository

Runner's Hi 프로젝트입니다.  
안드로이드 스튜디오와 코틀린을 기반으로 개발하며, 백엔드(Spring)와 연동하여 회원관리, 러닝 기록 저장, 랭킹 서비스를 제공합니다

---

## 1. 프로젝트 개요

- **앱 이름**: RunnersHi
- **플랫폼**: Android
- **주요 목적**
    - 러닝 기록 측정
    - 러닝 히스토리 조회 및 통계 제공
    - 랭킹 조회 기능 서비스 제공

---

## 2. 기술 스택

- **언어**: Kotlin
- **IDE**: Android Studio
- **UI**: 미정
- **아키텍처 패턴**: MVVM / Clean Architecture

---

## 3. 주요 기능

- 회원가입 / 로그인
- 러닝 시작 / 일시정지 / 종료
- 실시간 러닝 정보 표시 (시간, 거리, 페이스 등)
- 러닝 기록 저장 및 히스토리 목록 조회
- 러닝 상세 화면 (지도, 구간별 페이스 등)
- 설정 / 프로필 편집
- 랭킹 조회 기능
